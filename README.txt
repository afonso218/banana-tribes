# Banana Tribes #

################################### SINGLE PLAYER ###################################

BT Cliente possui duas classes que permitem correr a applet, que são: 
*Start.class ou Engine.class (dentro do package Engine).

Na pasta BT Cliente possui:
- Ficheiro HTML para correr a applet (no SO Mac OX é necessário configuração para visualizar);
- Ficheiro .jar para chamar a applet sem acesso ao código


################################ MULTIPLAYER PLAYER ################################

Necessário correr o BT Server para que o SINGLE PLAYER possa aceder ao MULTIPLAYER.
